# SMBMountManager

SMBMountManager 是一款專為 macOS 使用者設計的輕量級 SMB（Server Message Block）掛載管理工具。它解決了系統內建掛載功能不穩定、斷網時 Finder 卡頓以及無法根據網路環境自動切換掛載點等問題。

## 🌟 主要功能

### 1. 自動掛載與生命週期管理
- **開機啟動**：應用程式啟動時會根據設定自動嘗試掛載伺服器。
- **安全卸載**：當您退出應用程式時，系統會自動且安全地卸載所有掛載點，確保資源正確釋放。

### 2. 智慧監測與自動修復
- **狀態監控**：背景持續監測各掛載點的連線性及反應速度。
- **斷線重連**：若偵測到掛載點失效（例如伺服器重啟或網路暫時中斷），程式會自動嘗試重新建立連線。
- **防止卡頓**：針對斷網導致的磁碟掛載點「卡死」，提供強制解除掛載並重啟的機制，避免 Finder 失去反應。

### 3. 基於 Wi-Fi SSID 的連線規則
- **特定網路掛載**：您可以為每個掛載點設定「允許的 Wi-Fi SSID」。例如，您可以設定公司伺服器僅在連接辦公室 Wi-Fi 時才掛載，避免在咖啡廳或家中使用時產生不必要的連線錯誤。

### 4. 高安全性憑證管理
- **Keychain 整合**：密碼安全地儲存在 macOS 內建的 Keychain（鑰匙圈）中，不以明文儲存。

### 5. 優化 UI/UX
- **選單列即時狀態**：透過 Menu Bar 圖示即可掌握所有伺服器的連接狀態（連線中、已掛載、連線錯誤）。
- **進階連接測試**：新增掛載點前提供完整的連線測試，包含 Ping (ICMP)、連接埠 (TCP 445) 及 憑證驗證，並提供詳細的錯誤提示。

## 🚀 安裝與開發

本項目使用 Swift 開發，您可以直接使用 Xcode 開啟並編譯。

1. **複製項目**：
   ```bash
   git clone https://github.com/imstevelin/SMBMountManager.git
   ```
2. **開發環境**：
   - macOS 14.0+
   - Xcode 15.0+
3. **編譯**：
   開啟 `SMBMountManager.xcodeproj` 或 `project.yml`（由 XcodeGen 管理），然後點擊 Run。

## 🛠️ 技術規格

- **架構**：SwiftUI + ObservableObject 原生架構。
- **網路**：使用 `CoreWLAN` 進行 SSID 偵測，`Combine` 進行狀態同步。
- **掛載機制**：捨棄傳統的 `launchd` Agent，改由 App 內的單獨引擎 (`MountEngine`) 進行更精細的控制。
